
# MADIN NURIS

Aplikasi presensi murid dan guru berbasis Flutter.

Fitur:
- Presensi murid harian
- Presensi guru per jam
- Rekap bulanan otomatis
- Persentase kehadiran
- Ranking guru & kelas
- Edit data presensi
- Tambah/edit/hapus nama

Cara menjalankan:
1. Install Flutter
2. Jalankan:
   flutter pub get
   flutter run
